LeetCode No 14 Longest Common Prefix

1. 题目

Write a function to find the longest common prefix string amongst an array of strings.

2. 思路

如：“abcd”、“abefh”、“absyz”则其最长公共前缀为”ab”；再如：“abcdefg”、“abcfg”、“gabcdest”则其最长公共前缀为空。通过举例说明可知，这里的最长公共前缀指的是从每个字符串的第一个位置开始，若都相同，则匹配下一个，直到出现一个不相同或者某个字符串完结为止。

LeetCode No 13 Roman to Integer

1. 题目

Given a roman numeral, convert it to an integer.
Input is guaranteed to be within the range from 1 to 3999.

2. 思路

计数规则：

  1. 若干相同数字连写表示的数是这些罗马数字的和，如 III=3；
  2. 小数字在大数字前面表示的数是用大数字减去小数字，如 IV＝4；
  3. 小数字在大数字后面表示的数是用大数字加上小数字，如 VI=6;

LeetCode No 11 Container With Most Water

1. 题目

Given n non-negative integers a1, a2, …, an, where each represents a point at coordinate (i, ai). n vertical lines are drawn such that the two endpoints of line i is at (i, ai) and (i, 0). Find two lines, which together with x-axis forms a container, such that the container contains the most water.

2. 思路

2.1 暴力解决

第一个想到的就是用两个for循环穷举所有的情况，结果超时了…

2.2 两端靠拢

从两端开始向中间靠拢，如果左端线段短于右端，那么左端右移，反之右端左移，直到左右两端移到中间重合，记录这个过程中每一次组成木桶的容积，返回其中最大的。

LeetCode No 6 ZigZag Conversion

1. 题目

The string “PAYPALISHIRING” is written in a zigzag pattern on a given number of rows like this: (you may want to display this pattern in a fixed font for better legibility)

P A H N
A P L S I I G
Y I R

And then read line by line: “PAHNAPLSIIGYIR”
Write the code that will take a string and make this conversion given a number of rows:

string convert(string text, int nRows);

convert(“PAYPALISHIRING”, 3) should return “PAHNAPLSIIGYIR”.

2. 思路

2.1

比较直观的解法是，用一个字符串数组 string[row] 来存储每一行，最后一拼接就是最终结果。用一个step表示正向还是反向，即上图中从第一行到最后一行还是最后一行到第一行。

2.2

网上解法
发现所有行的重复周期都是 2nRows-2
对于首行和末行之间的行，还会额外重复一次，重复的这一次距离本周期起始字符的距离是2nRows-2-2*i
